HTML : HyperText Markup Language
웹에서 사용하는 문서

WWW : Word Wide Web
Web은 링크가 걸린 문서(HTML)를 주고 받아서 정보를 교환하는 서비스

Web Server : Server는 서비스를 제공하는 자 ex) 네이버, 구글, 다음 등 Web Client : Client는 서비스를 이용하는 자

Web Browser : HTML문서를 해석해서 화면에 띄어주는 도구
즉, 클라이언트가 웹 서비스를 이용하기 위해서 필요한 프로그램
종류 : Chrome, Edge, FireFox, Opera, Whale, 삼성 인터넷 등
